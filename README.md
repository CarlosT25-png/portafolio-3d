## Run Locally

Clone the project

```bash
  git clone https://github.com/CarlosT25-png/portafolio-3d.git
```

```bash
  cd 3d-portafolio
  cd 3d-portafolio
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm dev
```
